使用Hugo+GithubPages零成本搭建自己的博客

Feb 6, 2025

看完本文你会了解：如何利用hugo搭建博客，如何将博客部署在githubpage上

安装hugo

mac环境下使用如下命令安装hugo：

    brew install hugo

可以输入如下命令验证hugo是否成功安装：

    hugo version

生成博客

安装完成后，就可以输入以下命令创建博客项目，myblog为你的博客项目名，可以自己来定：

    hugo new site myblog

创建成功后，进入myblog目录下操作
